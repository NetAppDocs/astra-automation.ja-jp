= このリリースについて
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
このサイトのドキュメントでは、 Astra Control REST API 、および 2021 年 12 月（ 21.12 ）にリリースされる関連する自動化テクノロジについて説明しています。特に、このリリースの REST API は、対応する 21.12 リリースの Astra Control Center と Astra Control Service に含まれています。
