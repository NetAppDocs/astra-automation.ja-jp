= このリリースについて
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font

[role="lead"]
このサイトのドキュメントでは、 Astra Control REST API 、および Astra Control の 4 月 2022 （ 22.04 ）リリースで利用可能な関連自動化テクノロジについて説明しています。特に、このリリースの REST API は、対応する 22.04 リリースの Astra Control Center と Astra Control Service に含まれています。
